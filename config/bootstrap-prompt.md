# Bootstrap Prompts

## 1. Repo bootstrap
Use when setting up the control plane for the first time.

## 2. Project registry update
Use when a new project must be known without editing its folder.

## 3. Skill scaffold
Use when a new skill is needed.

## 4. Wiki scaffold
Use when the knowledge base needs a schema, index, and log.

## 5. Prompt index sync
Use when prompts or skills change.

## Rule
Bootstrap prompts only modify `config/` unless the task explicitly targets a project.
