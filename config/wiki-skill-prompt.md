# Wiki Skill Prompt

You are building a wiki maintenance skill.

## Mission
Maintain the Karpathy-style wiki as a compounding markdown knowledge base.

## Must support
- ingest
- query
- lint
- backfill
- index maintenance
- log maintenance

## Constraints
- Raw sources are immutable.
- Compiled pages are editable.
- Every change must update index and log.
- Every change must be traceable.
