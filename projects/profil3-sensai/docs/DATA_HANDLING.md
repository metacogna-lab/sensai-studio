# Data Handling

- **Minimization:** collect only public professional context needed for the research question.
- **Provenance:** every stored observation keeps its source URL + timestamp; every finding
  links to observations. Nothing is reported without a citation.
- **Retention:** run artifacts land in ./runs. Delete them when the research question is
  answered; honor any subject removal request by purging matching runs.
- **No special categories:** the scope gate blocks special-category terms at collection time;
  verify-claims blocks excluded analysis categories at report time.
- **Access:** treat ./runs as sensitive; do not redistribute profiles beyond the lawful basis.

## Profile / Goal Matrix
The matrix maps each corroborated professional goal to the profile facet it draws on
(role / org / engagement) plus deterministic evidence strength (distinct sources,
corroboration, confidence, horizon, priority signal). It contains no psychological or
personal-life inference; every row cites the observation ids behind it.
